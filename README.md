# Movie Recommender

If you want to run the notebook cells again, keep the following in mind:
- Download the dataset `rotten_tomatoes_movie_reviews.csv` from the following link, and keep it in the same folder as the notebook (Note: all dataset(s) used here belong to their respective owners):
https://www.kaggle.com/datasets/andrezaza/clapper-massive-rotten-tomatoes-movies-and-reviews

- Running the paragraph generation cells requires a Paid Google API Key for the paragraph generation to run (since google has removed the all free tokens).

To run the streamlit version of the application, do the following in your terminal:
```
pip install streamlit
streamlit run app.py
```
